A simple **Machine Learning web application** built with **Streamlit** that predicts employee salary based on **position level** using **Support Vector Regression (SVR)**.

The goal of this project is to demonstrate how a trained **machine learning model** can be integrated into a **user-friendly web interface**.

## 📊 Features

* Predict salary based on **position level**
* Interactive **Streamlit UI**
* **Support Vector Regression (SVR)** model with **RBF kernel**
* Visualization of **actual salary data vs predicted curve**
* Beginner-friendly ML project

## 📈 How It Works

1. The dataset contains **position levels and corresponding salaries**.
2. Data is **scaled using StandardScaler**.
3. A **Support Vector Regression (SVR)** model is trained using an **RBF kernel**.
4. Users choose a **position level**, and the model predicts the **expected salary**.
5. The app visualizes the **actual data points and SVR prediction curve**.
